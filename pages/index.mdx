# Orang Asli Millennials

Hi, Orang Asli Millennials adalah satu laman web untuk memberi inspirasi dalam mendokumentasikan warisan budaya, adat resam, cerita dongeng dan sejarah masyarakat Orang Asli di Semenanjung Malaysia.

Oleh itu, Orang Asli Millennials telah mewujudkan platform Khazanah Orang Asli untuk membukukan hasil hasil khazanah supaya dapat menjadi rujukan dan tatapan pada generasi akan datang.

Di dalam zaman COVID-19 yang serba mencabar ini, Orang Asli Millennials menggunakan aplikasi laman web untuk memudahkan orang ramai untuk mendapatkan rujukan.

Kami amat mengalakkan masyarakat orang asli untuk menyumbangkan idea dan bahan bahan untuk laman web ini.

Jika anda mempunyai karya karya menarik, sila hubungi social media kami di bawah.
